# AI Text-to-Image Generator

A modern, web-based application that generates images from text prompts using AI. Built with Flask and the Pollinations.ai API, featuring a premium glassmorphism UI.

## Features

- **Free & Unlimited Generation**: Uses the Pollinations.ai API (no API key required).
- **Fast Performance**: Generates images in seconds.
- **Advanced Controls**:
  - Custom Width & Height (using intuitive sliders).
  - Seed Control (for reproducible results).
- **History System**: 
  - Automatically saves the last 10 generated images.
  - Sidebar for easy access to past creations.
  - One-click restore of settings from history.
  - Persistent storage using browser `localStorage`.
- **Modern UI/UX**:
  - "Glassmorphism" design (blur effects, transparency).
  - Smooth animations (hover effects, button shines, loading states).
  - Responsive layout for various screen sizes.
- **Download**: Instantly download your generated masterpieces.

## Technology Stack

- **Backend**: Python (Flask)
- **Frontend**: HTML5, CSS3 (Variables, Flexbox/Grid), JavaScript (Vanilla)
- **API**: [Pollinations.ai](https://pollinations.ai/) (Stable Diffusion compatible)

## Setup & Running

1. **Prerequisites**: Ensure you have Python installed.

2. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
   *(Note: Main dependencies are `flask` and `requests`).*

3. **Run the Application**:
   ```bash
   python app.py
   ```

4. **Access**: Open your browser and navigate to `http://127.0.0.1:5000`.

## API Usage

This project uses the Pollinations.ai API, which is a free, URL-based image generation service. 
- **Endpoint**: `https://image.pollinations.ai/prompt/{prompt}`
- **Parameters**: `width`, `height`, `seed`, `nologo`
